branching

Branching di GitHub dilakukan menggunakan Git, yang mana Anda dapat membuat, 
menghapus, dan mengelola cabang (branch) melalui antarmuka baris perintah Git atau 
menggunakan fitur yang disediakan oleh platform GitHub. 

Fungsi Utama: Membuat cabang baru dari cabang utama atau cabang lain dalam repositori Git.

Manfaat:
- Memisahkan pengembangan fitur atau perbaikan bug ke dalam unit terpisah, yang memudahkan pengelolaan dan pengujian.
- Mengizinkan pengembang untuk bekerja secara terpisah tanpa mengganggu proyek utama, 
serta memungkinkan untuk mencoba berbagai ide atau eksperimen tanpa risiko merusak kode yang ada.

CONTOH: git branch <nama branch>
